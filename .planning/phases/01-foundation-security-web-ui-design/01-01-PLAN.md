    **Design System (index.css with Tailwind CSS 4):**

    Configure `frontend/src/index.css` using Tailwind CSS 4 CSS-first config:
    - `@import "tailwindcss";`
    - Define `@theme` block with CSS custom properties for shadcn/ui compatibility
    - Font family: Inter (geometric sans-serif per user decision)
    - Light theme (default): white background (#ffffff), near-black foreground (#09090b), black primary, red accent (#dc2626 — the "8" red from the logo). Neutral grays (zinc scale) for borders, muted elements, secondary text
    - Dark theme (`.dark` class): near-black background (#09090b), white foreground (#fafafa), white primary, same red accent. Dark zinc grays for cards, borders, muted elements
    - Red accent used SPARINGLY — only for logo highlight, destructive actions, important badges. NOT for primary buttons or links (per locked decision)
    - Primary buttons use neutral black (light) / white (dark) per Vercel Dashboard pattern
    - Rounded corners: `--radius: 0.5rem` (rounded & soft per decision)
    - Generous spacing throughout (spacious & airy density per decision)
    - Subtle transitions: `transition-colors duration-200` on interactive elements

    **shadcn/ui Components:**
