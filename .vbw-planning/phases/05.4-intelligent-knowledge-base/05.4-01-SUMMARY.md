## What Was Built

Zone-aware knowledge base data layer for the Intelligent KB. Evolved TemplateMapping with zone tracking, repetition counts, and correction history. Added BlueprintPattern model for structural pattern storage (loop/conditional/group markers) and StyleHint model for boilerplate detection. Service layer now supports confidence decay on corrections (0.7x), confidence boost on confirmations (+0.1 capped at 1.0), zone-filtered queries, and style-based filtering -- all backward compatible with existing callers.
