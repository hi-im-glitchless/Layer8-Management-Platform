## What Was Built

Structured KB-driven prompt generation replacing the flat few-shot section with zone-grouped context blocks. The analysis request now queries the KB for zone-grouped mappings, blueprints, boilerplate styles, and repetition data. When a template type has no KB data, cross-type fallback queries other types with a 0.7x confidence penalty. The prompt builder emits Zone Distribution, Blueprints, and Confidence Notes subsections with a dynamic 15-25% token budget (1500 cap). Boilerplate paragraph styles from StyleHint data are filtered from the doc structure section. On download, persistMappingsToKB() stores zone-enriched mappings, detects blueprints, and records style hints.
