## What Was Built

- HtmlReportPreview: sandboxed iframe with srcdoc, postMessage text selection bridge, ResizeObserver auto-sizing, empty state placeholder
- EntityMappingTable: 4-column table (Original Value, Placeholder, Entity Type, Actions) with inline type dropdown, delete button, truncation tooltips, manual badge, sticky header
- EntityPopover: floating popover for text-selection-based entity creation with type dropdown, duplicate/length validation, viewport-clamped positioning, dismiss on Escape/click-outside
- Shared entityTypes.ts with 15 entity type constants (PERSON through CUSTOM)
- StepSanitizeReview rewritten: full-width HTML preview, toggleable 60/40 split with mapping table, collapsible metadata accordion, optimistic entity CRUD via useUpdateEntityMappings
- SanitizationDiffView and DenyListEditor deleted, barrel exports updated
