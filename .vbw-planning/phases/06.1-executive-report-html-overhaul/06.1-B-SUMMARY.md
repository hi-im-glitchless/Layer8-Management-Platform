## What Was Built

- HTML-centric session model replacing DOCX-era fields (uploadedHtml, sanitizedHtml, entityMappings, entityCounterMap, supplementaryText, generatedHtml, chartConfigs)
- HTML text node sanitizer that walks DOM via node-html-parser, calls Presidio per text node, injects entity span tags with data attributes, and maintains session-scoped incrementing placeholders
- Upload pipeline rewritten to DOCX -> mammoth HTML -> sanitize HTML -> extract supplementary in one auto-triggered flow
- Routes updated: /sanitize and /update-deny-list removed, /update-entity-mappings added, /upload returns sanitizedHtml + entityMappings, /download serves PDF
- Frontend types/API/hooks aligned to new backend shape with EntityMapping type and useUpdateEntityMappings hook
