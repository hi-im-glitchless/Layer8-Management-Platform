## What Was Built

- StepReview rewritten: PdfPreview replaced with HtmlReportPreview iframe, de-sanitize toggle (Show Real Values / Show Sanitized) performs frontend-only placeholder replacement using entity mappings, chat corrections refetch session to update HTML preview
- StepDownload simplified to PDF-only: single Download PDF button with loading state, calls POST /api/report/download-pdf endpoint, no DOCX option
- convertHtmlToPdf() synchronous function in pdfQueue.ts sends HTML to Gotenberg Chromium endpoint with A4 format, zero margins, printBackground, and 3s waitDelay for Chart.js
- POST /api/report/download-pdf endpoint: validates session, applies server-side de-sanitization, converts HTML to PDF via Gotenberg, streams PDF buffer with correct headers
- ReportWizardShell step inference updated: Review step accessible when generatedHtml exists (no PDF dependency), sanitize-review checks sanitizedHtml
- Dead code removed: getReportDownloadPath, useReportPreviewStatus, getPreviewStatus, test_chart_renderer.py
