## Objective
Create the new StepVerify component that replaces StepAnalysis as the core verification step in the 4-step wizard. Shows the placeholder-styled PDF (Jinja expressions visible with light blue backgrounds) on the left and a chat panel on the right. Users review auto-mapped placeholders, select text for corrections using 5.2's numbered selection pattern, and click "Approve & Continue" when satisfied. Implements Decisions #3 (reuse 5.2 selection infra), #8 (unlimited iterations), #9 (approve triggers GW data render), #12 (chat always visible), #13 (bad auto-maps shown as-is), #14 (coverage counter shows placeholder count).

## Tasks

### Task 1: Placeholder Preview Backend Proxy
**File(s):** `backend/src/services/templateAdapter.ts`, `backend/src/routes/templateAdapter.ts`, `frontend/src/features/adapter/api.ts`, `frontend/src/features/adapter/hooks.ts`
**Action:** Modify
**Details:**
- In templateAdapter.ts, add `generatePlaceholderPreview(wizardState: WizardState): Promise<WizardState>`:
  1. Read adapted DOCX from wizardState.adaptation.appliedDocxPath (the post-insertion DOCX)
  2. Convert to base64
  3. POST to Python /adapter/placeholder-preview with adapted_base64, template_type, language
  4. Save annotated DOCX to disk (same DOCUMENTS_DIR pattern)
  5. Queue PDF conversion via addPdfConversionJob()
  6. Update wizard state with new annotatedPreview data: pdfJobId, placeholder list, placeholder count
  7. Return updated state
- In routes/templateAdapter.ts, add `POST /api/adapter/placeholder-preview`:
  - Zod body: `{ sessionId: z.string().uuid() }`
  - Verify session exists and has adaptation.appliedDocxPath (auto-map completed)
  - Call generatePlaceholderPreview(state)
  - Return `{ pdfJobId, placeholders, placeholderCount }`
- In api.ts, add to adapterApi:
  ```typescript
  async requestPlaceholderPreview(sessionId: string): Promise<PlaceholderPreviewResponse> {
    return apiClient<PlaceholderPreviewResponse>('/api/adapter/placeholder-preview', {
      method: 'POST',
      body: JSON.stringify({ sessionId }),
    })
  }
  ```
- In types.ts, add:
  ```typescript
  export interface PlaceholderInfo {
    paragraphIndex: number
    placeholderText: string
    gwField: string
  }
  export interface PlaceholderPreviewResponse {
    pdfJobId: string
    placeholders: PlaceholderInfo[]
    placeholderCount: number
  }
  ```
- In hooks.ts, add `usePlaceholderPreview()` mutation hook
**Commit:** `feat(adapter): add placeholder preview proxy endpoint and frontend API`
