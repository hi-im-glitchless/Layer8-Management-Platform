## Objective

Replace all user-visible "Layer8" text with "AI Template Regenerator" across the entire application. Logo image filenames are kept as-is (internal assets). This is a straightforward find-and-replace scoped to specific known locations.

## Context

- `@frontend/src/routes/Dashboard.tsx:10` -- "Welcome to Layer8. Your security reporting hub."
- `@frontend/src/routes/Login.tsx:90,105` -- two `alt="Layer8"` on logo images
- `@frontend/src/components/layout/Sidebar.tsx:104` -- `alt="Layer8"` on logo
- `@frontend/src/components/auth/OnboardingWizard.tsx:84,122,167` -- alt text, heading, body text
- `@frontend/src/index.css:51,86` -- CSS comments mentioning "Layer8"
- `@backend/src/services/auth.ts:51` -- TOTP QR code issuer label

## Tasks

**Scope:** `frontend/src/routes/Dashboard.tsx`, `frontend/src/components/layout/Sidebar.tsx`, `frontend/src/routes/Login.tsx`

**Action:**
- Dashboard.tsx line 10: `"Welcome to Layer8. Your security reporting hub."` → `"Welcome to AI Template Regenerator"`
- Sidebar.tsx line 104: `alt="Layer8"` → `alt="AI Template Regenerator"`
- Login.tsx line 90: `alt="Layer8"` → `alt="AI Template Regenerator"`
- Login.tsx line 105: `alt="Layer8"` → `alt="AI Template Regenerator"`

**Acceptance:** Dashboard shows "Welcome to AI Template Regenerator". Sidebar and Login logo alt text updated. No "Layer8" text in these 3 files.

### Task 3: Update CSS comments and backend TOTP issuer

**Scope:** `frontend/src/index.css`, `backend/src/services/auth.ts`

**Action:**
- index.css line 51: `/* Layer8 Light Theme: ... */` → `/* AI Template Regenerator Light Theme: ... */`
- index.css line 86: `/* Layer8 Dark Theme: ... */` → `/* AI Template Regenerator Dark Theme: ... */`

**Acceptance:** CSS comments updated. TOTP QR codes generated after this change show "AI Template Regenerator" as the issuer.

### Task 4: Final branding verification sweep

**Scope:** Entire `frontend/src/` and `backend/src/` directories

**Action:** Run a comprehensive case-insensitive search for any remaining "Layer8" references:
- `grep -ri "layer8" frontend/src/ backend/src/ --include="*.tsx" --include="*.ts" --include="*.css" --include="*.json"`
- Verify all user-visible instances are replaced
- Logo filenames (`layer8_logo_dark.png`, `layer8_logo_white.jpg`) are expected and should NOT be renamed

**Acceptance:** No user-visible "Layer8" text remains. Logo filenames are unchanged. The only remaining "layer8" strings are in image file paths (expected).
