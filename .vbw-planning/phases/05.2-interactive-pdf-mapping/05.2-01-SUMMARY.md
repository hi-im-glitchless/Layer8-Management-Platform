## What Was Built

Selection state management and interactive PDF text selection for Phase 5.2 interactive mapping. Users can select text on the PDF viewer, selections are auto-numbered with stable IDs, and numbered badges appear as overlays with status-based coloring (blue/green/orange). Toast notifications confirm each addition.

## Files Modified

- `frontend/src/features/adapter/types.ts` -- SelectionEntry, SelectionStatus, SelectionAction, SelectionBoundingRect, BatchMappingResult types
- `frontend/src/features/adapter/hooks.ts` -- useSelectionState hook with useReducer, auto-numbering, status tracking, computed counters, toast feedback
- `frontend/src/features/adapter/components/InteractivePdfViewer.tsx` -- New component wrapping PdfPreview with mouseup text capture, coordinate extraction, and numbered badge overlay with tooltips
