## What Was Built

- POST /adapter/document-structure Python endpoint that parses DOCX and returns all paragraphs including empty/whitespace-only ones with metadata (index, truncated text, heading level, is_empty, style name)
- GET /api/adapter/document-structure/:sessionId Node.js proxy with Zod response validation, snake-to-camelCase transform, and session caching to avoid re-parsing
- Frontend API client, TanStack Query hook (staleTime=Infinity), and ParagraphInfo/DocumentStructureResponse types
- StructureBrowser collapsible sidebar component: searchable paragraph list, empty/whitespace labels, heading level indicators, keyboard (Escape), animated slide-in, skeleton loading state
