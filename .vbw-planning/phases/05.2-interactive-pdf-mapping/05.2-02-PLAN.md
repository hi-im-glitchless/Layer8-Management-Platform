## Objective
Build the document structure browser sidebar that lists all DOCX paragraphs (including blank/invisible ones not visible in the PDF render). Users can select paragraphs from this sidebar to add them as numbered selections. This covers acceptance criteria #2 (structure browser sidebar for blank/invisible paragraphs). The backend endpoint parses the DOCX and returns paragraph metadata; the frontend renders a collapsible sidebar overlay.

## Context
- @sanitization-service/app/services/docx_parser.py -- Existing DocxParserService.parse() returns DocxStructure with paragraphs list
- @sanitization-service/app/models/docx.py -- DocxParagraph model has text, style_name, heading_level, alignment, runs
- @sanitization-service/app/routes/adapter.py -- FastAPI adapter routes, add new endpoint here
- @sanitization-service/app/models/adapter.py -- Pydantic models for adapter, add request/response models
- @backend/src/routes/templateAdapter.ts -- Node.js Express routes, add proxy endpoint
- @backend/src/services/wizardState.ts -- Has template_base64 in wizard state for proxying to Python
- Decision #2: Collapsible sidebar in PDF viewer -- toggle button on toolbar opens sidebar overlay

## Tasks

### Task 1: Python Document Structure Endpoint
**File(s):** `sanitization-service/app/models/adapter.py`, `sanitization-service/app/routes/adapter.py`
**Action:** Modify
**Details:**
- Add Pydantic models to `adapter.py`:
  ```python
  class DocumentStructureRequest(BaseModel):
      template_base64: str

  class ParagraphInfo(BaseModel):
      paragraph_index: int
      text: str  # truncated to 200 chars
      heading_level: int | None = None
      is_empty: bool
      style_name: str | None = None

  class DocumentStructureResponse(BaseModel):
      paragraphs: list[ParagraphInfo]
      total_count: int
      empty_count: int
  ```
- Add `POST /adapter/document-structure` endpoint to `adapter.py`:
  1. Decode base64 template to bytes
  2. Parse via DocxParserService().parse(doc_bytes)
  3. Map each paragraph to ParagraphInfo: truncate text to 200 chars, set is_empty=True if text.strip() is empty
  4. Return DocumentStructureResponse with paragraphs list, total count, and empty count
  5. Pydantic validation on input, 422 on invalid base64
**Commit:** `feat(adapter): add POST /adapter/document-structure endpoint for paragraph listing`

### Task 2: Node.js Proxy Endpoint
**File(s):** `backend/src/routes/templateAdapter.ts`
**Action:** Modify
**Details:**
- Add `GET /api/adapter/document-structure/:sessionId` endpoint:
  1. Validate sessionId via Zod (reuse existing sessionIdParamSchema)
  2. Require auth, verify session ownership
  3. Get wizard state, verify session has template file (base64)
  4. POST to Python `${sanitizerUrl}/adapter/document-structure` with `{ template_base64 }`
  5. Return Python response directly (paragraphs, total_count, empty_count)
  6. Cache in wizard state to avoid re-parsing on subsequent requests (add `documentStructure` field)
- Add Zod response validation for type safety
- Audit log event: `adapter.document_structure`
**Commit:** `feat(adapter): add document structure proxy endpoint with session caching`

### Task 4: StructureBrowser Sidebar Component
**File(s):** `frontend/src/features/adapter/components/StructureBrowser.tsx`
**Action:** Create
**Details:**
- Props: `{ sessionId: string; isOpen: boolean; onToggle: () => void; onSelectParagraph: (paragraphIndex: number, text: string) => void }`
- Uses `useDocumentStructure(sessionId)` hook to fetch paragraph list
- Renders as a fixed-width sidebar overlay (w-80) on the right side of the PDF viewer area
- Search input at top filters paragraphs by text content (case-insensitive)
- Paragraph list with scroll:
  - Each item shows: `#{index}` badge, heading level indicator (H1-H9), truncated text
  - Empty paragraphs: show `[Empty paragraph]` in muted italic with style name if available
  - Whitespace-only paragraphs: show `[Whitespace]` label
  - Clickable: calls `onSelectParagraph(paragraphIndex, text)` which parent wires to addSelection
- Loading state: skeleton list
- Empty state: "No paragraphs found"
- Toggle button: uses `PanelRightOpen`/`PanelRightClose` icons from lucide-react
- Keyboard: Escape closes sidebar
- Animated slide-in via Tailwind `transition-transform duration-200`
**Commit:** `feat(adapter-ui): implement StructureBrowser collapsible sidebar with search and paragraph selection`
