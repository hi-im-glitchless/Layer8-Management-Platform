## What Was Built

- Tabbed Step 2 layout with Preview (default) and Mappings tabs, Chat stays on right panel
- Preview tab renders annotated PDF via PdfPreview with coverage summary bar showing "N/M fields mapped (X% coverage)"
- Green (Mapped) and yellow (Gap) color legend badges in preview header
- Auto-triggered annotated preview generation after analysis completes
- Regenerate Preview button refreshes annotated PDF after inline edits
- "Preview outdated" yellow dot indicator on Preview tab when edits are pending
- Page reload restoration via useCachedAnnotatedPreview hook
- Enhanced MappingTable with gap rows (yellow background, [Gap] badge) below mapped entries
- Inline editing for gap rows: click GW Field cell to type, Enter/Escape to confirm/cancel
- MarkerType dropdown (text/paragraph_rt/run_rt) during gap row editing
- ParagraphPicker searchable popover triggered by "+ Add Missing Section" button
- Two-step paragraph picker: search/select paragraph, then enter GW field and marker type
- Optimistic local state updates with automatic revert on backend API failure
- Backend sync via useUpdateMapping deriving editedEntries/addedEntries from plan diff
- Popover UI component (shadcn wrapper for radix-ui Popover primitive)
