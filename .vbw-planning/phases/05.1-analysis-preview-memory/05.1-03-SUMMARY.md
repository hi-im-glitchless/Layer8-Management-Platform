## What Was Built

Backend integration wiring the Python annotation service and knowledge base into the Node.js adapter pipeline. Four capabilities: annotated preview endpoint (Python /annotate + Gotenberg PDF), KB persistence hook on download (fire-and-forget), mapping update endpoint for inline edits, and KB query integration in analyzeTemplate() for few-shot examples. 23 new tests across 2 test files.

- **WizardState extension** -- WizardAnnotatedPreview interface with pdfJobId, tooltipData, unmappedParagraphs, gapSummary. Added to WizardState with defaults and deep merge support.
- **generateAnnotatedPreview()** -- POSTs to Python /adapter/annotate with template + mapping plan, saves annotated DOCX to disk, queues PDF conversion via Gotenberg, updates wizard state with annotation metadata.
- **analyzeTemplate() KB query** -- Queries queryFewShotExamples() before calling Python /analyze, converts to snake_case few_shot_examples array. Graceful degradation on DB error (empty array, analysis continues).
- **POST /api/adapter/annotated-preview** -- Returns 202 with pdfJobId + tooltipData + unmappedParagraphs + gapSummary. Validates session is in analysis step.
- **GET /api/adapter/annotated-preview/:sessionId** -- Returns cached annotation data with PDF job status for page reload.
- **POST /api/adapter/update-mapping** -- Merges editedEntries (by sectionIndex, confidence=1.0) and addedEntries (from paragraph picker, confidence=1.0) into mapping plan. Zod validated.
- **Download KB hook** -- GET /download/:sessionId fires persistMappingsToKB() after stream starts. Non-blocking (.catch pattern). Audit log 'adapter.kb_persist'. Updates step to 'download'.
- **persistMappingsToKB()** -- Converts mapping plan entries to KB format, calls bulkUpsertMappings(). Skips null/empty plans. Errors logged, never thrown.
- **23 tests** -- 14 annotation/mapping tests (preview metadata, edit/add logic, KB query integration, Zod validation) + 9 download/KB tests (persistence hook, graceful degradation, audit log, usageCount increment).
