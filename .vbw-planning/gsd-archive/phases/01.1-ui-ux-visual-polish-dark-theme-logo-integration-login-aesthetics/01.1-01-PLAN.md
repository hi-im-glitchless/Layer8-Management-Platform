  <action>
Replace both `:root` (light) and `.dark` color palettes in `frontend/src/index.css` with a new OKLCH color system. Keep all existing `@theme inline` mappings and `@layer base` rules — only modify the CSS custom property values inside `:root` and `.dark`.

**Dark theme direction (VS Code/Discord-like):**
- `--background`: Deep near-black, ~oklch(0.16 0.005 260) (NOT pure black — slight cool tint, ~#1a1b1e)
- `--foreground`: Near-white, oklch(0.98 0 0) (~#fafafa)
- `--card`: Slightly elevated, ~oklch(0.20 0.005 260) (~#25262b)
- `--popover`: Same as card or slightly lighter, ~oklch(0.22 0.005 260)
- `--primary`: White on dark for primary buttons, oklch(0.985 0 0)
- `--primary-foreground`: Dark text on white buttons, oklch(0.16 0 0)
- `--secondary`: Subtle surface, ~oklch(0.25 0.005 260)
- `--secondary-foreground`: Near-white
- `--muted`: Slightly lifted surface ~oklch(0.25 0.005 260)
- `--muted-foreground`: Readable secondary text ~oklch(0.71 0 0) — must pass 4.5:1 on --background
- `--accent`: Cool blue, ~oklch(0.65 0.16 250) (~#4E9FFF)
- `--accent-foreground`: White text on blue, oklch(0.98 0 0)
- `--destructive`: Red for danger, ~oklch(0.65 0.2 25) (bright enough to read on dark)
- `--border`: Subtle, oklch(1 0 0 / 8%)
- `--input`: Slightly more visible than border, oklch(1 0 0 / 12%)
- `--ring`: Cool blue for focus rings, same as accent ~oklch(0.65 0.16 250)
- Sidebar colors: `--sidebar` slightly different from background (e.g., oklch(0.18 0.005 260)), sidebar accent matches the blue accent
- `--sidebar-accent`: Use the cool blue with low opacity for active backgrounds, ~oklch(0.65 0.16 250 / 15%)
- `--sidebar-accent-foreground`: Near-white

**Light theme direction (clean, polished, matching cool-blue accent):**
- `--background`: Pure white oklch(1 0 0)
- `--foreground`: Near-black oklch(0.145 0 0)
- `--card`: White oklch(1 0 0)
- `--primary`: Cool blue accent as primary in light mode, ~oklch(0.55 0.16 250) (darker blue for readability on white)
- `--primary-foreground`: White text on blue buttons
- `--secondary`: Light gray ~oklch(0.97 0.005 260)
- `--muted`: Light gray oklch(0.97 0.005 260)
- `--muted-foreground`: Medium gray ~oklch(0.55 0 0) — must pass 4.5:1 on white
- `--accent`: Cool blue ~oklch(0.55 0.16 250)
- `--accent-foreground`: White
- `--destructive`: Red ~oklch(0.55 0.2 25)
- `--border`: Light gray oklch(0.92 0 0)
- `--ring`: Cool blue, same as accent
- Sidebar: White background, blue accent highlights

**Chart colors:** Update chart colors to use blues, teals, and complementary hues instead of reds. Use a harmonious palette that works with the cool blue direction.

IMPORTANT: Keep the exact same CSS custom property names. Only change the OKLCH values. The `@theme inline` block must remain unchanged. Do NOT remove the `@custom-variant dark` line or the `@layer base` section.
