<objective>
Create comprehensive test suite for the sanitization pipeline using TDD approach: synthetic pentest report data, unit tests for each custom recognizer, deny list tests, mapping operator tests, and round-trip integration tests.

Purpose: Sanitization correctness is critical -- false negatives leak PII to the LLM, false positives over-redact and reduce LLM output quality. The TDD approach (RED -> GREEN -> REFACTOR) ensures each recognizer handles edge cases that are easy to miss: version strings falsely matching as IPs, product names matching as hostnames, and mixed-language content.

Output: Passing test suite covering all recognizers, deny list, mapping operator, and end-to-end round-trip.
</objective>

    **Recognizer tests (test_recognizers.py):**
    Cases: input text -> expected entity detections

    IP Recognizer:
    - "Server at 10.1.2.50 was vulnerable" -> detects 10.1.2.50 as IP_ADDR
    - "OpenSSH 8.2.1 running on host" -> does NOT detect 8.2.1
    - "Apache/2.4.51 installed" -> does NOT detect 2.4.51
    - "127.0.0.1 is localhost" -> does NOT detect (localhost excluded)
    - "192.0.2.1 documentation range" -> does NOT detect (RFC5737)
    - "Target: 10.1.2.50/24" -> detects IP with CIDR

    Hostname Recognizer:
    - "Connected to dc01.corp.local" -> detects dc01.corp.local as HOSTNAME
    - "Found srv-web01.internal" -> detects srv-web01.internal
    - "Visit google.com for info" -> does NOT detect (not internal TLD)

    AD Object Recognizer:
    - "CN=John Doe,OU=IT,DC=corp,DC=local" -> detects as AD_OBJECT
    - "Modified OU=Finance,DC=globaltech,DC=internal" -> detects as AD_OBJECT

    Network Path Recognizer:
    - "Access \\\\fileserver\\share\\docs" -> detects as NETWORK_PATH
    - "smb://fileserver/share" -> detects as NETWORK_PATH

    Domain Recognizer:
    - "Client website globaltech-corp.com" -> detects as DOMAIN
    - "github.com is well known" -> does NOT detect (excluded)

    **Deny list tests (test_deny_list.py):**
    - "Contact Acme Corp for details" with deny_list=["Acme Corp"] -> matches "Acme Corp"
    - "ACME CORP is the client" -> matches (case insensitive)
    - "AcmeticSoft is different" with deny_list=["Acme"] -> does NOT match (word boundary)
    - Empty deny list -> no matches
