Output: A runnable Python microservice at `sanitization-service/` that starts, loads models, and responds to health checks.

  <action>
    Create the `sanitization-service/` directory at the project root (sibling to `backend/` and `frontend/`).

    **sanitization-service/app/config.py:**
    - Use Pydantic BaseSettings for configuration
    - Fields: `host` (default "0.0.0.0"), `port` (default 8000), `redis_url` (default "redis://localhost:6379"), `log_level` (default "info"), `spacy_models` (list, default ["en_core_web_lg", "pt_core_news_lg"]), `default_confidence_threshold` (float, default 0.5)
    - Load from environment variables with `SANITIZER_` prefix (e.g., `SANITIZER_PORT=8000`)

    **sanitization-service/app/main.py:**
    - Create FastAPI app with title "Layer8 Sanitization Service"
    - On startup event (`@app.on_event("startup")` or lifespan context manager -- prefer lifespan as it's the modern FastAPI pattern), load spaCy models into a global `nlp_models` dict keyed by language code ("en", "pt")
    - Wrap model loading in try/except, log progress ("Loading en_core_web_lg...", "Loaded 2/2 models")
    - Set `models_loaded` flag accessible to health endpoint
    - Mount health router
    - Add CORS middleware allowing all origins (will be called only from Node backend on same network)
    - Include `app/models/` for shared state (nlp_models dict, models_loaded flag) -- use a simple module-level state object or app.state

    **sanitization-service/app/health.py:**
    - FastAPI APIRouter with `/health` GET endpoint
    - Returns 200 with `{"status": "healthy", "models_loaded": true, "supported_languages": ["en", "pt"]}` when models are loaded
    - Returns 503 with `{"status": "unavailable", "models_loaded": false, "supported_languages": []}` when models are NOT loaded
    - Access model state from app.state or shared module

    **sanitization-service/app/models/request.py:**
    - `SanitizeRequest`: text (str), session_id (str), deny_list_terms (list[str], optional, default []), entities (list[str] | None, optional -- filter to specific entity types), language (str | None, optional -- override auto-detection)
    - `DesanitizeRequest`: text (str), session_id (str)

    **sanitization-service/app/models/response.py:**
    - `DetectedEntity`: entity_type (str), start (int), end (int), score (float), text (str), placeholder (str)
    - `SanitizeResponse`: sanitized_text (str), entities (list[DetectedEntity]), language (str), entity_counts (dict[str, int]), warning (str | None -- e.g., "No PII detected")
    - `DesanitizeResponse`: text (str), complete (bool -- true if all placeholders resolved), unresolved_placeholders (list[str])
    - `HealthResponse`: status (str), models_loaded (bool), supported_languages (list[str])

    **sanitization-service/Dockerfile:**
    - Base image: `python:3.12-slim`
    - Install system dependencies needed by spaCy (gcc, build-essential) in a single RUN layer, then clean up apt cache
    - Set workdir to `/app`
    - Copy requirements.txt first (Docker layer caching)
    - `pip install --no-cache-dir -r requirements.txt`
    - Download spaCy models: `python -m spacy download en_core_web_lg && python -m spacy download pt_core_news_lg`
    - Copy application code
    - Create non-root user: `useradd -m appuser && chown -R appuser /app` then `USER appuser`
    - Expose port 8000
    - CMD: `uvicorn app.main:app --host 0.0.0.0 --port 8000`
    - Add HEALTHCHECK: `HEALTHCHECK --interval=10s --timeout=5s --retries=3 --start-period=15s CMD curl -f http://localhost:8000/health || exit 1`

    **sanitization-service/.dockerignore:**
